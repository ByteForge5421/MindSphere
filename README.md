# 🧠 MindSphere — AI-Driven Mood & Community Engagement Platform

MindSphere is a full-stack behavioral analytics platform designed to help users track mood patterns, maintain journals, and engage with a supportive community while enabling product-level analytics to monitor user behavior and engagement trends.

---

## 🚀 Key Features

### 👤 User Experience Features
- Mood tracking and journaling workflows
- Community interaction system
- Real-time data synchronization
- Personalized engagement insights

---

## 📊 Product Analytics Capabilities

MindSphere was designed with a **product analytics-first approach** to simulate real-world product data tracking systems.

### ✅ Event Tracking System
- Captures key product metrics such as:
  - Daily Active Users (DAU)
  - Session duration
  - Feature adoption rates
  - Engagement frequency
- Implements event instrumentation similar to workflows used in:
  - Google Analytics
  - Mixpanel

---

### 📈 Analytics Dashboard
Interactive dashboards to monitor:

- User retention trends
- Funnel drop-off analysis
- Cohort behavior tracking
- User segmentation insights
- Feature usage KPIs

These dashboards enable **data-driven product decision making**.

---

### 🧪 Experimentation Framework

Implemented an A/B testing system to evaluate product improvements:

- Tested reminder notification strategies
- Performed statistical analysis of engagement impact
- Achieved a **30% increase in 7-day retention**
